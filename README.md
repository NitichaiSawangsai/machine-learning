
# Install

```
$ brew install pyenv poetry
$ pyenv install 3.9
$ pyenv global 3.9
$ python -V
$ which node 
$ brew uninstall --ignore-dependencies node
$ nvm ls
$ nvm install 18.20
$ npm i
$ poetry env list
$ poetry env remove ${poetry env list {0}}
$ poetry env use 3.9
$ poetry shell
$ python --version
```


# Add Package
```
$ npm add-package ${name Package} 
```


# Run Project

### set ENV
 create file .env
```    
VERSION=1.0.0
```

### run
```
$ poetry shell
$ npm run dev
```
